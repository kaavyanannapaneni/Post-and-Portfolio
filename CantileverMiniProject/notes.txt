Styling text

1. Changing fonts with **font-family**
   font-family: Arial, sans-serif;

   - the fonts visible may depend on the fonts available in the system (system fonts)
   - use fallback font family at the end of the font stack 
   - to check view in dev tools, computed 

   To know the common fonts present in mac and windows
   https://www.cssfontstack.com/

2. font-size, font-weight, font-style

font-size: 12px;

font-weight: 100
100 - 900

NOTE: check if the font you choose has differnt weights designed sometimes 
      only (for example) 100, 400 may be visible and when you give 
      font-weight: 900 
      it defaults to the closest font-weight available that is 400 

font-style: normal;
font-style: italic;

3. text-alignment 

text-alignment: left;
left, right, justify

justify - align left and right, with space in between 

4. line-height, letter-spacing, word-spacing 

font-size: 16px
line-height: 2;
without units, it is a multiplier 
2 *  the font-size
2 * 16px = 32px is the line height 

letter-spacing: 10px;
spacing between letters

word-spacing: 16px;
spacing between words

5. Adding custom fonts with Google fonts 
   select the fonts and link it in html or css file 
   html file is better option 

6. Creating **text shadows**
many formats are there refer mdn doc 

x-offset, y-offset, blur-radius, color
text-shadow: 2px 2px 2px red;
